# 🧮 Calculadora de IMC

Uma calculadora de **Índice de Massa Corporal (IMC)** desenvolvida com **ReactJS**, utilizando **Vite** para o build e **TailwindCSS** para a estilização. Simples, responsiva e direta ao ponto.

## ✨ Tecnologias

- ⚛️ ReactJS
- ⚡ Vite
- 🎨 TailwindCSS
